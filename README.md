# Certify-Punchit-App

Este es un proyecto frontend de código abierto de la aplicacion de certificados de PUNCH IT

### Tecnologías

- React 🤩
- React-router-dom 🎲
- Redux 🧠
- Firebase 🔥
- Styled-Components 💅🏼

### Iniciar el proyecto

Agregá tus credenciales de firebase en firebase/firebase-config.js

```bash
npm start
```
